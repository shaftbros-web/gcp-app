# GCP 認定試験対策クイズ

Google Cloud 認定試験の対策問題をカテゴリ別に学習できるインタラクティブなクイズWebアプリケーションです。

## 特徴

- **複数カテゴリ対応**: 「ネットワーク」「ストレージとデータベース」「セキュリティとIAM」など、分野別に学習可能。
- **全問通しモード**: 全問正解するまで、間違えた問題が繰り返し出題されます（サドンデス形式）。
- **弱点克服モード**: 過去の正答率に基づいて、一定の閾値以下の苦手な問題のみを集中的に学習できます。
- **学習記録の保存**: ブラウザの LocalStorage を利用し、カテゴリごとに正解・不正解の統計を記録します。
- **即時フィードバック**: 回答後にその場で正解・不正解判定と、各選択肢ごとの詳しい解説が表示されます。

## 使用技術

- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- Node.js (問題データ抽出スクリプト用)

## 使い方

1. リポジトリをクローンするか、ファイルをダウンロードします。
2. ブラウザで `index.html` を開きます。
3. 学習したいカテゴリを選択し、クイズモードを選んで学習を開始してください。

## 問題データの生成方法

追加の問題ファイル (`*_b.txt`) を同じディレクトリに配置し、以下のコマンドを実行することで、問題データ (`quiz_data.js`) を更新できます。

```bash
node parse_quiz_multi.js
```
